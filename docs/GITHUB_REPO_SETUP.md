# GitHub Repo Setup

Use this guide to polish the GitHub repository metadata before sharing Spektra with reviewers or recruiters.

## Description

Recommended repository description:

> A full-stack media tracking platform for films, series, games, and books built with React, Laravel, FastAPI, PostgreSQL, Redis, and Docker.

## Topics

Add these repository topics:

- `react`
- `typescript`
- `vite`
- `laravel`
- `fastapi`
- `postgresql`
- `redis`
- `docker`
- `media-tracker`
- `pwa`
- `fullstack`

## Website

Set the GitHub website field to the deployed demo URL once available.

Recommended steps:

1. Deploy the frontend, API, and worker using [DEPLOYMENT.md](DEPLOYMENT.md).
2. Run the smoke checks in [RELEASE_CHECKLIST.md](RELEASE_CHECKLIST.md).
3. Add the public frontend URL to the GitHub repository website field.
4. Replace the README demo placeholder with the same URL.

Do not link to localhost, temporary private previews, or deployments that require secrets or personal credentials.
